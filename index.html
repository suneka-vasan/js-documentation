var, let and const

  • We can redeclare the variable which is declared with var, but we can't redeclare the variable which is declared with let or const.
  • If we think the value of the variable might change, use let. const value can't be changed.
  • var - redeclaring a variable inside a block will also redeclare the variable outside the block.
  • let - redeclaring a variable inside a block will not redeclare the variable outside the block.
  • const - value can't be reassigned or redeclared.

var x = "name";

var x = "firstName";

Objects

  • Objects are variables too. But it contain many values.
  • The values are written as pairs(name: value).
  • It is a common practice to declare objects with the const keyword.
  • JavaScript objects are containers for named values called properties.
const person = {firstName:"John", lastName:"Doe", age:46};
//Objects use names to access its "members". In this example, person.firstName returns John

Arrays

An array is a special variable, which hold more than one value.

const person = ["John", "Doe", 46];

JavaScript Array Methods
Method Description
toString() converts an array to a string of (comma separated) array values.
join() same as tostring() method, but in addition we can specify the separator.
pop() removes the last element from an array.
push() adds a new element to an array (at the end).
shift() removes the first element from an array.
unshift() adds a new element to an array (at the beginning).
splice() adds a new elements to an array(more than 1). 
const fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.splice(2, 0, "Lemon", "Kiwi");
//above 2 defines where new element should be added. 0 defines how many elements should be removed. Lemon, Kiwi defines the new element to be added.
OUTPUT: Banana,Orange,Lemon,Kiwi,Apple,Mango
slice() slices out a piece of an array into a new array.
The slice() method creates a new array.
The slice() method does not remove any elements from the source array. 
const fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
const citrus = fruits.slice(2);
document.getElementById("demo").innerHTML = fruits + "<br>" + citrus;
OUTPUT:
Banana,Orange,Lemon,Apple,Mango
Lemon,Apple,Mango
delete() Array elements can be deleted using the JavaScript operator delete.
Using delete leaves undefined holes in the array.
Use pop() or shift() instead. 
const fruits = ["Banana", "Orange", "Apple", "Mango"];
delete fruits[0];
document.getElementById("deletedemo").innerHTML = "The first fruit is: " + fruits[0];
OUTPUT: The first fruit is: undefined
concat() reates a new array by merging (concatenating) existing arrays.
The concat() method does not change the existing arrays. It always returns a new array.
const myGirls = ["Cecilie", "Lone"];
const myBoys = ["Emil", "Tobias", "Linus"];
const myChildren = myGirls.concat(myBoys);

document.getElementById("demo").innerHTML = myChildren;
OUTPUT: Cecilie,Lone,Emil,Tobias,Linus

every() It check if all array values pass a test.
const numbers = [45, 4, 9, 16, 25];
let allOver18 = numbers.every(myFunction);

document.getElementById("demo").innerHTML = "All over 18 is " + allOver18;

function myFunction(value, index, array) {
return value > 18;
}
// it checks the all array values are above than 18.

OUTPUT: All over 18 is false
some() It check if all array values pass a test.
const numbers = [45, 4, 9, 16, 25];
let someOver18 = numbers.some(myFunction);

document.getElementById("demo").innerHTML = "Some over 18 is " + someOver18;

function myFunction(value, index, array) {
return value > 18;
}
// it checks some of the array values are above than 18.

OUTPUT: Some over 18 is true
indexOf() It searches an array for an element value and returns its position.
const fruits = ["Apple", "Orange", "Apple", "Mango"];
let position = fruits.indexOf("Apple") + 1;

document.getElementById("demo").innerHTML = "Apple is found in position " + position;
// If the item is present more than once, it returns the position of the first occurrence.

OUTPUT: Apple is found in position 1
lastIndexOf() same as Array.indexOf(), but returns the position of the last occurrence of the specified element.

find() It returns the value of the first array element that passes a test function.
