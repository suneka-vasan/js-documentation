var, let and const

  • We can redeclare the variable which is declared with var, but we can't redeclare the variable which is declared with let or const.
  • If we think the value of the variable might change, use let. const value can't be changed.
  • var - redeclaring a variable inside a block will also redeclare the variable outside the block.
  • let - redeclaring a variable inside a block will not redeclare the variable outside the block.
  • const - value can't be reassigned or redeclared.

JavaScript if, else, and else if

Conditional statements are used to perform different actions based on different conditions.

  • if - a block of code to be executed, if a specified condition is true
  • else - a block of code to be executed, if the same condition is false
  • else if - a new condition to test, if the first condition is false
if (condition1) {
// block of code to be executed if condition1 is true
} else if (condition2) {
// block of code to be executed if the condition1 is false and condition2 is true
} else {
// block of code to be executed if the condition1 is false and condition2 is false
}

JavaScript Break and Continue

The break statement "jumps out" of a loop.
The continue statement "jumps over" one iteration in the loop.


let text = "";
for (let i = 0; i < 10; i++) { 
    if (i===3) { break; } 
    text +="The number is " + i + "<br>" ; 
}
    document.getElementById("demo").innerHTML=text;

OUTPUT:
The number is 0
The number is 1
The number is 2
The Continue Statement

continue statement breaks one iteration (in the loop), if a specified condition occurs, and continues with the next iteration in the loop.


let text = "";
for (let i = 0; i < 5; i++) { 
    if (i===3) { continue; } 
    text +="The number is " + i + "<br>" ; 
}
    document.getElementById("demo").innerHTML=text;

OUTPUT:
The number is 0
The number is 1
The number is 2
The number is 4

JavaScript Sets

A JavaScript Set is a collection of unique values.
Each value can only occur once in a Set.

Method Description
new Set() Creates a new Set
add() Adds a new element to the Set
delete() Removes an element from a Set
has() Returns true if a value exists in the Set
forEach() Invokes a callback for each element in the Set
values() Returns an iterator with all the values in a Set
How to Create a Set

You can create a JavaScript Set by:

  • Passing an Array to new Set()
  • Create a new Set and use add() to add values
  • Create a new Set and use add() to add variables

// Create a Set

const a = "a";

3

JavaScript typeof

In JavaScript there are 5 different data types that can contain values.

  • string
  • number
  • boolean
  • object
  • function

There are 6 types of objects.

  • Object
  • Date
  • Array
  • String
  • Number
  • Boolean

And 2 data types that cannot contain values.

  • null
  • undefined

You cannot use typeof to determine if a JavaScript object is an array (or a date).

Primitive Data

A primitive data value is a single simple data value with no additional properties and methods.


typeof "John" // Returns "string"
typeof 3.14 // Returns "number"
typeof true // Returns "boolean"
typeof false // Returns "boolean"
typeof px // Returns "undefined" (if px has no value)
Complex Data

The typeof operator can return one of two complex types:

  • function
  • object

The typeof operator returns "object" for objects, arrays, and null.
The typeof operator does not return "object" for functions.


typeof {name:'John', age:34} // Returns "object"
typeof [1,2,3,4] // Returns "object" (not "array", see note below)
typeof null // Returns "object"
typeof function myFunc(){} // Returns "function"

Note: The typeof operator returns "object" for arrays because in JavaScript arrays are objects.
